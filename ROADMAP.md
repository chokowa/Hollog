# Bocchi SNS 開発ロードマップ

現在の「ローカル完結型Webアプリ」としての強み（爆速な開発スピードと修正力）を最大限に活かしつつ、最終的に動画などの制限をなくした「完全なネイティブアプリ」へと進化させるための段階的なロードマップです。

## Phase 1: Webアプリ版の完成・極限のブラッシュアップ（実用版完了）
**目的：開発スピードの最速化とUI/UXの完成度向上**

Phase 1 は、ネイティブ化前の機能検証としては完了扱いにします。細部のUI/UX調整は、Android実機での操作感を見ながらPhase 2以降で継続します。

- [x] 複数画像の投稿・保存機能（ IndexedDB への配列保存 ）
- [x] 高品質な画像表示レイアウト（Xのようなグリッドクロップ、全画面画像ビューア）
- [x] 設定（Settings）画面の実装
  - [x] タグのサジェスト候補の追加・編集・削除機能
  - [x] タグ候補一覧（▼から開いた時のリスト）の並び替え機能
  - [x] テーマ切り替え、投稿済みカード表示設定
- [x] 検索・タグ絞り込みの基本機能
  - 本文・URL・OGP情報・タグを対象にした検索
  - タブ内タグ一覧とタグクリックによる絞り込み
- [x] カレンダー表示
  - 日付ごとの投稿確認
  - ポスト / クリップの絞り込み
- [x] タグ管理・一括タグ付け
  - 専用のタグ管理画面
  - 複数投稿へのタグ追加・置換
  - タグ更新時はタイムライン順を維持する
- [ ] UI/UX の完全な洗練（ネイティブ化後に実機操作を見ながら継続）
- [x] タイムライン画像のサムネイルキャッシュ化
  - [x] 投稿時に軽量サムネイルを生成・保存する
  - [x] タイムラインはサムネイル、画像ビューア/詳細は元画像を使う
  - 将来のネイティブ化では元画像を端末ファイル参照、サムネイルだけアプリ管理にする
- [ ] 未実装機能の補完
  - プロフィール機能は方針未定。自己紹介用途は不要な可能性が高く、削除も含めて検討する
  - 検索・タグ管理は基本実装済み。必要に応じて履歴、保存条件、タグ整理の操作感を追加調整する
- [ ] PWA（ホーム画面追加）による疑似アプリとしての運用テスト
  - [x] Web App Manifest の基本設定
  - [x] ホーム画面追加用アイコンの追加
  - [ ] ホーム画面追加の実機確認
  - [ ] オフライン動作やキャッシュ戦略の要否を判断

*※このフェーズでは、動画対応による容量問題やビルドエラーのトラブルを避け、Web技術のメリット（ホットリロード）をフル活用して機能を作り込みます。*

## Phase 2: ネイティブアプリ化の準備（中期）
**目的：Web技術からスマホ/PCネイティブ環境への橋渡し**

- [ ] アプリの基本機能が完全に安定した段階で移行を決定
- [x] Capacitor または Tauri などのネイティブ化フレームワークの導入・選定
  - Android優先のためCapacitorを採用
- [x] Android向けの最小ネイティブ起動準備
  - Next.js静的出力（`out/`）をCapacitorのWeb資産として同期
  - OGP取得は初期ネイティブ版では任意機能化
- [ ] GitHub + Vercel による本番環境（HTTPS）へのデプロイ
  - ブラウザのセキュリティ制限（クリップボードAPI等）を解除するためのHTTPS環境の確保。
  - PWAとしての完全なインストール機能の検証。
- [ ] ネイティブ環境でのビルドテスト（Android Studio / 将来Xcode）
  - [x] Android Debugビルド確認
  - [x] Android実機へインストールして起動
- [ ] Android実機での最小動作検証
  - 投稿・画像保存・再起動後のデータ保持
  - タイムライン、詳細、画像ビューア、タグ管理
- [ ] iOS 実機での動作検証

## Phase 3: ネイティブアプリ化の完了と制限の解放（最終）
**目的：OSの機能に直接アクセスできる最強のパーソナルアプリの実現**

- [ ] **動画投稿・再生機能の完全解放**
  - ネイティブのファイルシステム権限を利用し、アプリ内にデータを二重コピーすることなく、端末内の大容量動画をそのまま参照・再生する仕組みの実装。
- [ ] OSネイティブ機能の統合
  - スマホの「共有メニュー」から直接Bocchi SNSに保存できる機能の実装。
  - オフラインでの完全なネイティブ動作とパーフォーマンスの最適化。
- [ ] 完全版としてのリリース・日常運用開始
